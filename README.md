# 📊 RETAIL SALES DASHBOARD

## 📌 Overview

This project is an interactive **Retail Sales Dashboard** built using Streamlit. It analyzes retail sales data to provide insights into sales trends, customer behavior, and product performance.

---

## 🚀 Features

* 📦 Sales analysis by product category
* 👥 Customer analysis by gender
* 📈 Sales trends over time
* 🥧 Category distribution (pie chart)
* 🔥 Correlation heatmap
* 👥 Age group analysis
* 📅 Monthly sales trends
* 📥 Downloadable reports (CSV)

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Matplotlib
* Seaborn

---

## ▶️ Run Locally

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the app:

```
streamlit run app.py
```
